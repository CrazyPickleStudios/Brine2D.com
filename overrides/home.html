Code-first 2D game engine

Everything you need — scenes, ECS, audio, input, collision, particles, and UI — in one NuGet package. No editor. No visual tools. Just C#.

Get Started API Reference
What's included
🖼️
GPU Rendering
Hardware-accelerated via SDL3 GPU — Vulkan, Metal, and D3D12. Sprites, sprite sheets, cameras, particles, and render targets.
🧩
Hybrid ECS
Components for data, Behaviors for per-entity logic with DI, Systems for batch processing. One World per scene.
🎮
Input
Keyboard, mouse, and gamepad. Polling and event-driven. Input layers for UI-eats-input patterns.
🔊
Spatial Audio
Distance attenuation, stereo panning, configurable falloff. Audio source and listener ECS components included.
💥
Collision
AABB and circle colliders, spatial queries, collision events. Kinematic character controller built-in.
🎬
Animation
Clip-based animation with state machines, blend trees, layers, and direct Aseprite JSON import.
🗺️
Tilemaps
Native Tiled map loading and rendering. Multiple layers, tile properties, and object groups.
🪟
UI System
Buttons, sliders, text inputs, dialogs, scroll views, tooltips, and dropdowns — all integrated with ECS and DI.
📦
Asset Pipeline
Unified loader with ref-counted caching and typed manifests for parallel preloading. Assets release when the scene unloads.
